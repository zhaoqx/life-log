# 变更日志

## v1.0.0 (2025-10-14)

### 新增功能
- 手机信息采集功能，支持写文章和添加文件到OneNote和OneDrive
- Microsoft账户认证集成
- OneNote文章同步
- OneDrive文件和照片上传
- 基础配置系统，为未来分类和目录配置做好扩展准备

### 技术实现
- 使用Microsoft Graph API进行OneNote和OneDrive操作
- OAuth 2.0 认证流程
- Python实现，提供良好的跨平台支持

### 已知限制
- 本版本暂不支持自定义分类和目录配置
- 需要手动配置Microsoft应用程序凭据
